Live Demo

Project Description

The Flames Fantasy Game is an entertaining web app that calculates and reveals the relationship between two individuals based on their names. Users input their names, and the app determines their relationship status with options like "Friends," "Lovers," "Affectionate," "Married," "Enemies," and "Siblings". The game follows a set of predefined rules to make this determination. It's a simple yet enjoyable way to predict relationships in a playful manner.

My Role

Flames is a classic fantasy game where players can determine their "relationship fate" with another person based on their names. It's a fun and interactive web application that my friend and I developed collaboratively. My primary role in this project was as the frontend developer, responsible for designing and creating the user interface and establishing the frontend-backend connectivity.
