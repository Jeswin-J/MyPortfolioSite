Project Description

NameGen is my first Flutter project, a fun app that simplifies the process of generating random names. The user-friendly interface incorporates a side drawer navigation bar that allows quick access to various app components. The favourites option, which allows users to bookmark and return names of interest, is a remarkable feature. This clean and efficient Flutter app offers a pleasant experience for browsing a wide range of names and easily grouping personal preferences.

Key Features

Key features include an intuitive side drawer navigation bar for effortless app navigation. The app also incorporates a favorites option, allowing users to save and revisit names of interest. With its sleek design and user-friendly interface, NameGen provides a smooth and enjoyable mobile experience, making it easy to discover and organize a diverse array of names.
