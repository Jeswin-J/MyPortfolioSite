Project Description

A bus tracking system is an advanced technological solution meant to increase the efficiency and dependability of public transportation. This system allows users to monitor the current location and movement of buses on a dynamic map by utilising real-time GPS tracking and communication features. Passengers can obtain this information via Android app or Webapp, allowing them to better plan their journeys, reduce waiting times, and make informed decisions about their travel routes. By optimizing fleet management and boosting overall service quality this system benefits both passengers and transportation authorities.

My Role

As the frontend developer for our Real-Time Bus Tracking System, I focused on creating a user-friendly interfaces for Android and Web app using Java. My role involved implementing features like real-time bus updates, multi-language support, Wear OS Connectivity, SoS Feature and interactive maps. Collaborating closely with the team, I addressed challenges to ensure a smooth user experience. This project enhanced my skills in mobile app development and collaboration within a dynamic team.
